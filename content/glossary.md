# معجم مصطلحات — عبر المواد

مصطلحات عامة مفيدة لطلاب الثانوي الأدبي؛ المعاني مبسطة للمراجعة السريعة.

| المصطلح | المعنى المختصر |
|--------|-----------------|
| **منهج** | الخطة الدراسية المعتمدة رسمياً في المدرسة. |
| **وحدة دراسية** | مجموعة دروس تتناول موضوعاً واحداً ضمن المادة. |
| **مراجعة** | إعادة تنظيم المعلومات وتثبيتها قبل الامتحان. |
| **خط زمني** | ترتيب الأحداث من الأقدم للأحدث (أو العكس) لرؤية السياق. |
| **خرائط ذهنية** | رسم يربط الأفكار الفرعية بفكرة مركزية. |
| **نص مقالي** | سؤال يتطلب إجابة مكتوبة بعدة جمل أو فقرات. |
| **اختيار متعدد** | سؤال بعدة بدائل؛ واحدة صحيحة عادة. |
| **قاعدة نحوية** | قانون لبناء الجملة في اللغة (عربي أو أجنبي). |
| **بلاغة** | علوم الجمال اللغوي: المعاني والبيان والبديع. |
| **إحصاء وصفي** | وصف البيانات بجداول ورسوم ووسطيات دون استنتاج عن مجتمع كامل. |
| **احتمال** | قياس لفرض حدوث حدث ما ضمن فضاء عيّنة. |
| **Compréhension** | فهم المقروء (شائع في الإنجليزية والفرنسية). |
| **Tenses** | أزمنة الأفعال في الإنجليزية. |
| **Les temps** | أزمنة الأفعال في الفرنسية. |

أضف مصطلحاتك الخاصة من كتاب المدرسة في دفتر منفصل — هذا المعجم لا يغني عن المصطلحات الرسمية في منهجك.
